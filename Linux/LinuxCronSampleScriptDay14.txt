******************** SAMPLE SCRIPT FOR CRON ******************

🖥️ Scenario: Suppose we login to one server and we are at its home directory. Now create one script and add to cron job at same path that print some output every 5 minute.


🚀 Step 1: Decide kya automate karna : we want har 5 min mein ek script chale

📝 Step 2: Script create kar
	nano /home/ubuntu/test.sh

	#!/bin/bash
	echo "Cron chal raha hai" >> /home/ubuntu/test.log
	date >> /home/ubuntu/test.log
	
🔐 Step 3: Permission dena : without executable permission script won't run
	chmod +x /home/ubuntu/test.sh    

🧪 Step 4: Manual test (BEST PRACTICE)
	sh /home/ubuntu/test.sh

👉 check karo:
	cat /home/ubuntu/test.log
✔ agar output aa raha hai → script sahi hai

#Now till above we just create script, now to run it for a particular time, we take help of cron means add this script in crontab file.


⚙️ Step 5: Cron job add karo
	crontab -e
    👉 first time open hoga → editor choose karo (nano easy hai)

	*/5 * * * * /home/ubuntu/test.sh >> /home/ubuntu/cron.log 2>&1
	🧠 What above line do: */5 → har 5 min script run hogi, output + error → cron.log me save
	
	
✅ Step 6: Verify cron job
	crontab -l
	we will see this:*/5 * * * * /home/ubuntu/test.sh >> /home/ubuntu/cron.log 2>&1

🔍 Step 7: Check chal raha hai ya nahi
	Option 1: log check karo
	cat /home/ubuntu/test.log
	
	Option 2: cron logs
	grep CRON /var/log/syslog
	
Real Flow: 	
Login → Script banao → chmod +x → manually run → crontab -e → schedule likho → verify → logs check

🔍 How to know issue is there?

🔹 Step 1: Output log check karo (FIRST THING)
	cat /home/user/cron.log

	👉 Output:

	/bin/sh: 1: /home/user/wrong.sh: not found
	
	
🔹 Step 2: System logs check karo
	grep CRON /var/log/syslog

	👉 Dikhega: cron job run hui, par error aaya
	
🔹 Step 3: Script manually run karo
	sh /home/user/wrong.sh

	👉 Turant error milega: No such file or directory
	
	
🚀 How to fix ?
🔹 Step 1: Correct path find karo
	pwd
	ls

	👉 ya:

	find / -name script.sh 2>/dev/null
	
🔹 Step 2: Cron update karo
	crontab -e

	👉 Fix karo:

	* * * * * /home/user/correct.sh >> /home/user/cron.log 2>&1
	
🔹 Step 3: Logs clear karo
	> /home/user/cron.log
	
🔹 Step 4: Verify
cat /home/user/cron.log

👉 Ab error nahi aana chahiye

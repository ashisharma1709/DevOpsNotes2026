************************** LINUX UMASK CONCEPT *******************

Linux mein umask (user mask) ek permission control mechanism hai jo decide karta hai ki newly created files aur directories par by default kaunsi permissions remove hongi.

🔹 Simple : Jab bhi hum koi file ya directory create karte hain, system ek default permission set deta hai:
File ka default: 666 (rw-rw-rw-)
Directory ka default: 777 (rwxrwxrwx)

👉 Lekin umask in permissions ko restrict karta hai (minus karta hai)

🔹 Umask ka kaam kya hai?

Umask batata hai: "Kaunsi permissions DENY karni hai by default"

🔹 Example:

Agar umask = 022 hai:

📁 Directory ke liye:
777 - 022 = 755
(rwxr-xr-x)

📄 File ke liye:
666 - 022 = 644
(rw-r--r--)


Check current umask:

👉 umask

Set new umask:

👉 umask 027

🔹 Important Points

✔️ Sirf creation time pe apply hota hai
✔️ Existing file pe effect nahi ❌
✔️ Har user ka alag ho sakta hai
✔️ System security ke liye use hota hai


🔹 Check umask
umask
🔹 Set umask
umask 002
🔹 Permanent umask
echo "umask 002" >> ~/.bashrc


umask -S

👉 output:u=rwx,g=rwx,o=rx

👉 matlab:
owner → full
group → full
others → no write


👉 umask -S kya dikhata hai?

👉 Final permission nahi dikhata
👉 Ye dikhata hai:

“Default se kya remove nahi hua”

Step 1: Default file permission
666 → rw-rw-rw-

👉 Notice:

❗ file mein execute (x) hota hi nahi by default

Step 2: Umask = 0004
Remove → others ka read (r)

Step 3: Apply karo
rw-rw-rw-
      ↓ remove 'r' from others
rw-rw--w-

👉 Result:

-rw-rw--w-

❓ “umask -S mein o=wx kyu?”
umask -S
u=rwx,g=rwx,o=wx

👉 iska matlab:

owner → kuch bhi remove nahi
group → kuch bhi remove nahi
others → read remove kiya

👉 isliye bacha: wx


Directory case (777)
777 → rwxrwxrwx
umask 004 → remove 'r' from others
Result → rwxrwx-wx  ✅ (o=wx match)

File case (666)
666 → rw-rw-rw-
umask 004 → remove 'r' from others
Result → rw-rw--w-  ❗ (no x, because x tha hi nahi)


🔹 Given:umask 0077

🔹 Step-by-step

📄 File default: 
666 → rw-rw-rw-
❌ Umask kya hataega:
077 → group aur others se sab hata do

🔹 Final result:
rw-------  → 600

👉 Exact output:

-rw-------
